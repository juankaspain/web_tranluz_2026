# Vision de producto

## Posicionamiento

Tranluz debe presentarse como el partner tecnico que reduce riesgo operativo en obras electricas y de telecomunicaciones mediante suministro, alquiler, revision, formacion y trazabilidad.

La web nueva debe decir, en menos de 5 segundos:

- Que hace Tranluz.
- Para quien lo hace.
- Por que es fiable.
- Como pedir ayuda, presupuesto o documentacion.

## Promesa central

Equipos, servicio tecnico, alquiler, formacion y trazabilidad para trabajos electricos y de telecomunicaciones con estandar profesional.

## Principios de experiencia

1. Primero resolver, despues contar.
2. Menos folleto, mas herramienta de trabajo.
3. Cada pagina debe tener una accion clara.
4. Cada producto debe conectarse con servicio, alquiler, documentos y accesorios.
5. Cada servicio debe explicar proceso, alcance, resultado y contacto.
6. La web debe parecer un centro tecnico vivo, no un archivo de PDFs.

## Publicos principales

- Responsable de compras.
- Jefe de obra.
- Responsable de mantenimiento.
- Tecnico de prevencion.
- Responsable de formacion.
- Instalador electrico o telecom.
- Cliente recurrente que busca una referencia concreta.

## Propuesta diferencial

Tranluz no debe competir solo por precio. Debe competir por:

- Especializacion desde 1987.
- Conocimiento de obra.
- Relacion con fabricantes lideres.
- Servicio tecnico oficial y homologado.
- Alquiler de equipos de tendido.
- Plataforma Revisa para trazabilidad.
- Formacion tecnica para empresas.
- Catalogo amplio y asesoramiento experto.
