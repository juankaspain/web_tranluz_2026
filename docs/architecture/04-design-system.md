# Sistema visual

## Direccion creativa

Industrial premium, tecnico, fiable y preciso. La web debe sentirse como una herramienta profesional para empresas electricas y de telecomunicaciones.

## Rasgos visuales

- Fotografia real de producto, taller, obra y tecnicos.
- Fondos limpios y alto contraste.
- Tablas tecnicas claras.
- Iconografia funcional.
- Componentes densos, ordenados y escaneables.
- Pocos elementos decorativos.
- Lenguaje visual de precision, mantenimiento, seguridad y obra real.

## Tipografia

- Titulares: sans industrial sobria, peso 650-750.
- Texto: sans legible, peso 400-500.
- Datos tecnicos: monoespaciada solo para referencias, codigos y medidas.

## Componentes clave

- Header con buscador tecnico.
- Mega menu por productos, soluciones y servicios.
- Barra de acciones: presupuesto, telefono, catalogo.
- Fichas de producto.
- Tablas de especificaciones.
- Comparador de equipos.
- Selector de alquiler.
- Formularios por intencion.
- Bloques de documentos.
- Bloques de proceso.
- FAQs tecnicas.
- CTA de soporte.

## Criterios de calidad visual

- La home debe mostrar producto o trabajo real en el primer viewport.
- Las paginas de catalogo deben priorizar busqueda y filtros.
- Los servicios deben tener narrativa de proceso, no solo parrafos.
- Las descargas deben tener version, fecha, peso y categoria.
- Las paginas moviles deben conservar acciones principales visibles.
