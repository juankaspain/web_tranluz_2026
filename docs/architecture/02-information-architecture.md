## Home top tier

La home debe ser una mesa de control comercial, no una carta institucional.

Orden recomendado:

1. Hero con propuesta clara y buscador tecnico.
2. Accesos rapidos: comprar, alquilar, revisar, formar, soporte.
3. Categorias de producto.
4. Servicios diferenciales: ITS, Revisa, Rent Puller, Formacion.
5. Prueba de autoridad: desde 1987, fabricantes, certificaciones, experiencia.
6. Productos/equipos destacados.
7. Centro tecnico: catalogos, guias, manuales.
8. CTA final por departamento.

## Pagina de producto

Cada producto debe tener:

- Nombre claro y referencia.
- Imagenes reales.
- Estado: comprar, presupuesto o alquiler.
- Ficha tecnica estructurada.
- Aplicaciones.
- Compatibilidades y accesorios.
- Documentos descargables.
- Servicio tecnico asociado.
- Formacion o revision si aplica.
- Productos relacionados.
- CTA de presupuesto.

## Pagina de servicio

Cada servicio debe tener:

- Problema que resuelve.
- Para quien es.
- Alcance.
- Proceso paso a paso.
- Resultado entregable.
- Equipos cubiertos.
- Normativa o certificacion si aplica.
- Preguntas frecuentes.
- CTA especifico.
